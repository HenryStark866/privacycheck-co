Privacidad por Diseño

Principio Transversal

Evalúa si la privacidad se incorporó proactivamente como un requisito integral desde el inicio del diseño, no como un añadido posterior.

1

¿La privacidad y protección de datos se incorporó como requisito desde las primeras etapas del diseño del producto o proceso?

2

¿Se realizó o está planificado un Análisis de Impacto en la Privacidad (PIA / EIPD) para el producto o proceso bajo evaluación?

3

¿Se ha evaluado la necesidad de designar un Oficial o Delegado de Protección de Datos (DPO) dentro de la organización?

4

¿Se diseñaron procedimientos para gestionar transferencias internacionales de datos si el producto o servicio lo requiere?

Sí — Completamente implementado Parcialmente — En proceso o incompleto No — No implementado
